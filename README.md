# SHENKAR CAR - EX-03-DISTRIBUTED-CLOUD-SERVICES  

Heyyyyyy there dear bodek, welcome to **Shenkar Car Rental Agency** - the best in the Middle East!

This is **Exercise 03** for the **Distributed Cloud Services** course.  
I built the **frontend** using **React Functional Components** and **MUI (Material UI)**.  

### Note:  
There is **no backend yet** -  all data is stored in a **local JSON file**, so obiovsly changes **do not remain** after a page refresh...

---

## the stack i used: 

- **React** (Functional Components)  
- **React Router DOM** (Navigation & Routing)  
- **CSS Modules** (Component-specific styles)  
- **Material UI (MUI)** (For UI components)  
- **Local JSON Data** (Since there is no backend)  
- **Netlify** (For deployment)
